# Onboarding New Agent

## Goal

Allow a new Codex account, Claude session or engineer to resume work with minimal ambiguity.

## Read order

1. `README.md`
2. `docs/project-context.md`
3. `docs/engineering-playbook.md`
4. latest ADRs in `docs/adr/`
5. `vault/00-home/index.md`
6. relevant integration docs in `vault/05-integrations/`

## After reading

The new agent should summarize:

- current product objective
- current data architecture
- current deployment architecture
- current next milestone

If the agent cannot do that, the docs are incomplete and must be improved.
